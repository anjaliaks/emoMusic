emoMusic is a Emotion Based Music Recommendation System webapp which can guess your emotion through your facial expressions and can recommend songs accoringly.<br/>

# Tech Stack :

## Front-end
- HTML5
- CSS3
- Bootstrap

## Back-end
- Python

# Features

**📖 Multi-Page Layout**

**🎨 Styled with Bootstrap and Css**

**🧑 Detects emotion through facial expression**

**🎼 Recommends songs on YouTube according to your mood**
